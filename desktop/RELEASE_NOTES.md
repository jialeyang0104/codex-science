# Codex Science Desktop

Initial desktop release for macOS and Windows.

## Highlights

- Cross-platform Electron app for AI-assisted scientific research.
- Provider profiles for OpenAI, DeepSeek, and custom OpenAI-compatible APIs.
- Local encrypted API key storage through Electron safeStorage when available.
- Workflow router for literature review, PDF/document work, manuscript narrative, clinical protocols, indication dossiers, single-cell, omics pipelines, biomodels, remote compute, instrument conversion, and research strategy.
- Dense research workbench UI based on the `ui-ux-pro-max` design pass.

## Notes

- Providers are bring-your-own-key.
- Model IDs are configurable because provider model catalogs change over time.
- The app does not upload local files automatically; users decide what text or data to send to a provider.
