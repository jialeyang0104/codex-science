# Research Strategy Routing

Use this route for idea selection, project triage, stuck projects, grant framing, and strategic scientific decisions.

## Process

1. Ask for the shortest possible project statement if missing.
2. Identify the scientific question, decision point, and success criterion.
3. Separate novelty, feasibility, impact, data availability, and execution risk.
4. Build a decision tree with stop/go criteria.
5. Convert the strategy into the next concrete artifact: aim page, study plan, analysis plan, risk table, or experiment list.

## Specialist Skill

Prefer `scientific-problem-selection` when available.
